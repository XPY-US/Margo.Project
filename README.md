## 📦 Project Components

### 🧬 Margo Firmware
A state-of-the-art firmware image designed for the **M5Stack StickS3**. 
* **Optimized Performance:** Low-latency core for real-time operations.
* **Target Hardware:** ESP32-S3 (M5Stack StickS3).
* **Binary Path:** `MargoFirmware/dist/Margo-m5stack-sticks3.bin`.

### 🔥 Margo Burner
A cyber-minimalist desktop utility built to flash Margo firmware with zero friction.
* **Modern Aesthetic:** Borderless "Bento Box" layout with glass-morphic cards.
* **High-Speed Flashing:** Support for up to **921600 baud**.
* **Fullscreen Support:** Use `F11` to toggle and `Esc` to exit.
